API 사용법

디버그

/hello

return "hello"

hello 를 출력합니다.

/query
?test-query={value}

return "쿼리 : {value}"

입력한 쿼리를 출력합니다.

시간표

/schedular
?date={date}
&grade={grade}
&class={class}

{"code":리턴코드, "header":{"dateFrom":데이터 시작날짜, "dateTo":데이터 종료날짜}, "data":[{"weekday":요일(숫자), "weekday_str":요일(문자), "period":교시, "item":수업 이름}]}

입력한 날이 포함된 주의 시간표를 json 형식으로 제공합니다.
서현고등학교 전용이며, 타 학교는 추후 지원할 예정입니다.

급식

/meal
?date={date}

{"code":리턴코드, "meal":급식 리스트, "cal":칼로리}

입력한 날의 급식 정보를 json으로 제공합니다.
서현고등학교 전용이며, 타 학교는 추후 지원할 예정입니다.

답지

/marker

준비중입니다.

문제집 등의 답지 정보를 json으로 제공합니다.
추후 제공할 예정입니다.
